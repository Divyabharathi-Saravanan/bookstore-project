# Bookstore Management System

## 📌 Project Description
This is a Spring Boot based Bookstore Management System.
It allows users to register, login, add books, edit books, manage cart, and view inventory.

## 🚀 Technologies Used
- Java
- Spring Boot
- Spring Security
- Thymeleaf
- MySQL / H2 Database
- Maven

## 📂 Project Structure
- src/main/java → Java source files
- src/main/resources/templates → HTML pages
- src/main/resources/static → Images & static files

## 🔐 Features
- User Registration & Login
- Secure Authentication
- Add / Edit / Delete Books
- Inventory Management
- Shopping Cart

## ▶️ How to Run
1. Clone the repository
2. Open in IntelliJ / Eclipse
3. Run BookstoreProjectApplication.java
4. Open browser: http://localhost:8081
